# Docs Index

当前文档目录按职责分成 4 类：

- `spec/`
  - 正式规格文档，例如 PRD、SAD、SRS。
- `design/`
  - 具体功能或机制的设计文档，例如 AI 推荐、项目进度、评分机制。
- `guides/`
  - 可复用的工程实践、迁移方案、构建说明、目录规范。
- `meta/`
  - 项目内部协作规则、临时待办、非正式管理类文档。
